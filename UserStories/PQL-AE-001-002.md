# EPIC ID: PQL-AE-001

**Name:** Add buildings

**Description:** As part of the core features of ThemePark@UCR, the system must allow administrative users to register new buildings  to keep track of the university's infrastructure. This includes the ability to add a completetely new building, and updating already existing buildings by editing their information.

## User Story

```gherkin
Feature: Edit information of buildings

  As an administrative user
  I want to edit the building information
  So that I can keep the university's infrastructure up to date.
```
## Acceptance Criteria

```gherkin
  Scenario: Edit all the building information
    Given I am on the administrative module of the system
    When access a building information
    Then an edit option should appear
    And I should be able to edit any information and get a confirmation message that it was succesfully edited
```

```gherkin
  Scenario: Edit the information of a building with invalid or missing information
    Given I am editing a building's information
    When leaving an information empty or adding invalid information
    Then an error message should appear
    And the information should not change
```

```gherkin
  Scenario: Edit the information of a non-existing building
    Given I am editing a building's information
    When I try to edit the information of a building that doesn't exist
    Then an error message should appear
```
