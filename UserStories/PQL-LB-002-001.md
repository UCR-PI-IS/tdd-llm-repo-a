# EPIC ID: PQL-LB-002
**Name:** List Buildings
**Description:** As part of ThemePark@UCR's core functionality, the system must allow for the listing of all registered buildings to facilitate navigation, query, and selection of information.

## User Story
```gherkin
Feature: List all buildings

As the ThemePark@UCR Administrator,
I want to be able to view the buildings in the system,
so that I can efficiently monitor and manage the university's infrastructure.
```
## Acceptance Criteria

```gherkin
  Scenario: Display all registered buildings
  Given the system has multiple buildings registered
  When a user navigates to the buildings section
  Then the system should display a list of all the buildings
```

```gherkin
  Scenario: No buildings registered in the system
  Given no building is registered
  When the user navigates to the buildings section
  Then the system should display an empty list
```

```gherkin
 Scenario: Large number of buildings
  Given there are many buildings in the system
  When the user accesses the list
  Then have a scroll system for usability
```
